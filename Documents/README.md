# Goals

The Goal is to build **frontend and Backend API** Using Rust as Programing Language.

## Phase 1

- The Backend will serve **Frontend** requests using a SQLite DataBase.
- Frontend will be WebAssembly **Web Client** using Yew rust framework.

## Phase 2

- Introduce SSL so we can move to **HTTPS**.
- Add **versioning** (Path Base).
- Add **Authentication** :
  - Backend and Frontend.
  - Oauth2 + Google.

## Phase 3

- Build a Yew Based **Web Client**:
  - Simple Client.
  - integrate Authentication.
  - Add More Pages.
- Polysh the Style UI and UX.
- API internal Documentation.

## Phase 4

- Add more ML based Solutions to the user Interface:
  - **Sentiment analyses**
  - **Translation from French to English**.
  - **Search Based Entities** (for exampel :Economical entities, compagnies by name...).
- **Add more Moroccan newspapers**.
- **User dashboard** with usage metrics.
- **Payment integration** (Stripe/PayPal).
